# MicroZim

A very simple Zim-Wiki plugin for Micro-editor.

## Installation

Copy the files file into ~/.config/micro/plug/microzim

# Default binding keys

* Alt+1 is heading 1
* Alt+2 is heading 2
* Alt+3 is heading 3
* Alt+4 is heading 4
* Alt+5 is heading 5
* Alt+t will render the selected paragraph as verbatim
